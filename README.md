#  Rendu Projet SCSS avec Vite.js

Ce projet a été réalisé avec **Vite.js** et **SCSS** dans le cadre d’un rendu d’évaluation.  
Il met en pratique l’utilisation des **mixins**, du **nesting** et d’une organisation claire des styles avec Sass.

##  Comment voir le projet

Pour visualiser correctement le rendu, il faut **lancer le projet avec Vite.js**.  
Vite permet de compiler le SCSS en CSS, de recharger la page automatiquement lors des modifications (hot reload) et d’optimiser le code pour la production.

###  Étapes à suivre :
1. Téléchargez ou clonez le projet :
   
   git clone https://github.com/RayaneLaidi/renduProjet_SCSS.git
   cd renduProjet_SCSS
   
Installez les dépendances nécessaires :

Copier le code
npm install

Lancez le serveur de développement :

Copier le code
npm run dev
Ouvrez le lien affiché dans le terminal (souvent http://localhost:5173/) pour voir le site.

 À propos du projet
Le projet utilise :

Vite.js pour le serveur local et la compilation rapide.

SCSS pour les styles avec nesting et mixins.

Une structure simple :
index.html
src/
├── style.scss
├── _mixins.scss
└── main.js
Vite surveille automatiquement le dossier src/, compile le SCSS et recharge la page instantanément à chaque modification.
